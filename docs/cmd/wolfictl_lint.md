## wolfictl lint

Lint the code

### Usage

```
wolfictl lint
```

### Synopsis

Lint the code

### Options

```
  -h, --help                    help for lint
  -l, --list                    prints the all of available rules and exits
      --skip-rule stringArray   list of rules to skip
  -v, --verbose                 verbose output
```
